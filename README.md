# Image to PDF Converter
A simple Python application that converts one or more images into a single PDF document.

# Features
- Convert images to PDF
- Merge multiple images into one PDF
- Supports JPG, JPEG, PNG, and other image formats
- Simple command-line interface

# Requirements
- Python 3.x
- Pillow (PIL)

# Installation
Install the required library:
pip install pillow

# How to Run
1. Place your image files in the project folder.
2. Open a terminal in the project directory.
3. Run:
image_to_pdf_converter.py
4. Enter image filenames separated by commas.
Example:
photo1.jpg, photo2.png, photo3.jpeg
5. The generated PDF will be saved as:
output.pdf

# Sample Output
